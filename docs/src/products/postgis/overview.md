# PostGIS Overview

## Why Choose this Product?

PostGIS brings advanced **spatial capabilities** to the industry-leading **PostgreSQL** database, allowing you to store, query, and analyze both spatial and non-spatial data in one system.

Built on open-source technologies and hosted for you in the cloud, this product delivers a complete, secure, and ready-to-use spatial database environment, no setup or server management required.

<div class="image-with-caption">
  <img src="../img/postgis-img-3.png" alt="PostGIS Benefits" style="width: 65%;">
</div>

- **Fully managed environment**

    Your hosted PostGIS instance comes preconfigured and maintained for you. No manual installation, version management, or infrastructure setup required. Everything runs securely in the cloud.

    <br>

- **Complete spatial database stack**

    Every hosted instance includes PostgreSQL, PostGIS, and pgAdmin, giving you all the tools needed to manage, query, and visualize your data through a browser-based interface.

    <br>

- **Scalable performance**

    Designed to handle increasing workloads, your hosted PostGIS instance can efficiently manage large datasets, simultaneous users, and demanding spatial operations.

    <br>

- **Seamless integration**

    Easily connect to external tools like QGIS or GeoServer, enabling you to use your hosted database as the backbone for both desktop and web GIS applications.

    <br>

- **Centralized data management**

    Keep your spatial and tabular datasets together in one place. Whether you’re running analytics, powering web maps, or managing enterprise data, PostGIS ensures consistency and accessibility across teams.

    <br>

- **Familiar SQL interface**

    Query spatial and non-spatial data using standard SQL syntax, enhanced with PostGIS functions for geometry creation, distance calculation, buffering, and more.

<br>
