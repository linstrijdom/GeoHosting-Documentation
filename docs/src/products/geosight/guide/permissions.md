# Managing Users & Permissions

GeoSight provides a flexible user and access management system that allows you to control who can view, edit, and manage data across the platform.

Permissions can be assigned at multiple levels, including datasets, indicators, and projects, giving you fine-grained control over collaboration and data security.

<br>

## Users & Groups

All user and permission management is handled from the <span class="ui-page-label">Users & Groups</span> section under <span class="ui-page-label">Access</span> in the Administrator Panel.

<br>

### Adding Users

You can create new users directly within GeoSight and assign them the appropriate access level. To add a user:

1. Navigate to <span class="ui-page-label">Access</span> → <span class="ui-page-label">Users & Groups</span>.

2. Click <span class="ui-generic-label">Create New User</span>.

3. Enter the required details (e.g email, credentials).

4. Assign a permission level.

5. Click <span class="ui-generic-label">Save</span> and share the login details with the user.

### Creating Groups

Groups allow you to manage multiple users at once by assigning shared access permissions. To create a group:

1. Navigate to <span class="ui-page-label">Users & Groups</span> → <span class="ui-page-label">Groups</span>.

2. Select <span class="ui-generic-label">Create New Group</span>.

3. Enter a group name.

4. Click <span class="ui-generic-label">Add Users</span>.

5. Search for and select users from the list.

6. Click <span class="ui-generic-label">Update Selection</span>, then <span class="ui-generic-label">Save</span>.

## Sharing & Access Control

GeoSight makes it easy to share datasets, indicators, and dashboards with other users. With flexible permission settings, you can control access at both the resource and project level.

<br>

### Sharing Resources

GeoSight allows you to share access to specific resources, including:

- Indicators
- Context Layers
- Projects
- Styles

<br>

To share a resource:

1. Open the item and click <span class="ui-generic-label">Share</span>.

2. Click <span class="ui-generic-label">Share to New User(s)</span>.

    <br>

    <div class="image-with-caption">
      <img src="../../img/geosight-img-28.png" alt="Share Resource">
      <div class="caption">
        Image credit: <a href="https://geosight.unicef.org/" target="_blank">GeoSight</a>
      </div>
    </div>

    <br>

3. Select the relevant users.

4. Choose a permission level:

    <br>

    <table class="my-table-style">
      <thead>
        <tr>
          <th>Permission</th>
          <th>Description</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>Read</td>
          <td>Allows users to view the resource.</td>
        </tr>
        <tr>
          <td>Write</td>
          <td>Allows users to edit and update the resource.</td>
        </tr>
        <tr>
          <td>Owner</td>
          <td>Full control, including deleting the resource.</td>
        </tr>
      </tbody>
    </table>

    <br>

5. Apply the changes to grant access.
